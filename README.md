ALE+
====

Features:

*   Very dark theme
*   Save using Ctrl + S
*   1, 4, 5 and custom snappings
*   Set value to anything (Right Alt for string-values)
*   Permanent clipboard
*   Find objects (Ctrl + F)
*   Download map as XML
*   Editor Manual by EaglePB2
